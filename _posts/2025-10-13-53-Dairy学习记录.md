## 2025.10.13 to 2025.10.17

1. 美国短期国债上涨，中长期国债下跌，并且国债的收益率曲线斜率收窄说明了什么？
   市场预期短期内美联储依然维持较高的利率并且未来美国经济放缓，货币政策会放松（比如降息）
   收益率曲线斜率收窄意味着市场对经济前景的看法趋于谨慎，既担忧短期高利率对经济的压制，又预期中长期政策会转向宽松以支撑经济。

2. 大牛市情况下，恒生为何选择私有化？
   恒生坏账率6.69%，退市后可以更灵活处理。并且香港市场在复苏周期，跟汇丰整合，恒生深耕的本地化业务和汇丰的全球业务结合，更利于企业发展。

3. 公司或者股票估值一般用可比或者DCF，为什么不选择GGM？

   可比法直接PE,PS，EV/EBITDA直接估值，bloomberg上拉一下数据马上就估值好了，反映了市场共识。

   DCF 拆业务，量化，DCF聚焦公司内在价值，适合深度研究

   GGM未来股息折现需要的前提太多，niche 工具而已，只适用极少数成熟高股息公司，比如中国移动这种,算出来目前处于低估区间。

4. 抓取基本面数据，构建估值模型做出基本面因子，这种策略是可行的吗？
   基本面决定了公司的长期价值，基本面因子可以用于中长期的持仓策略。但是基本面数据一般来自于公司的三表财务数据和季报年报等，估值建模也是需要基于这些数据来做。数据会有延迟，缺省，甚至少部分造假的情况。数据质量需要长期的维护，比较辛苦

5. 三表怎么看？

   | 报表名称       | 核心定位（回答的问题）                                       | 关键逻辑                                                     |
   | -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
   | **资产负债表** | 企业 "有什么"（资产）、"欠什么"（负债）、"剩什么"（股东权益） | 资产 = 负债 + 股东权益（静态平衡）<br />资产端（看 "优质资产" 占比）<br />负债端（看 "偿债压力"）<br />股东权益端（看 "家底厚度"） |
   | **利润表**     | 企业 "赚没赚钱"（收入 - 成本 = 利润）、"怎么赚的"            | 收入 - 成本费用 = 净利润（动态过程）<br />收入端（看 "收入含金量"）<br />费用端（看 "成本控制能力"）<br />利润端（看 "盈利真实性"） |
   | **现金流量表** | 企业 "钱从哪来"（现金流流入）、"钱花哪去"（现金流流出）      | 经营 + 投资 + 筹资现金流净额 = 现金净增加额<br />经营活动现金流净额<br />投资活动现金流净额<br /> 筹资活动现金流净额 |

   | 勾稽方向                | 具体对应关系                                                 | 健康标准（匹配表现）                      | 异常信号（不匹配风险）                                       |
   | ----------------------- | ------------------------------------------------------------ | ----------------------------------------- | ------------------------------------------------------------ |
   | 利润表 ↔ 现金流量表     | 现金流量表 "经营活动现金流净额" ≈ 利润表 "净利润" + 非付现费用（折旧 / 摊销） - 营运资金增加（应收账款 / 存货增长） | 经营现金流净额 ≥ 净利润（利润有现金支撑） | 经营现金流净额为负，但净利润为正（靠赊账虚增利润）           |
   | 资产负债表 ↔ 现金流量表 | 资产负债表 "货币资金期末数 - 货币资金期初数" = 现金流量表 "现金及现金等价物净增加额"（经营 + 投资 + 筹资净额之和） | 差额趋近于 0（无未披露的资金占用）        | 差额过大（如货币资金增 1 亿，但现金净增加额仅 5000 万，可能有受限资金未披露） |
   | 资产负债表 ↔ 利润表     | 资产负债表 "未分配利润期末数" = 未分配利润期初数 + 利润表 "净利润" - 当期分红（利润分配表中披露） | 数据完全相等（利润正常结转）              | 未分配利润增长远低于净利润，且无分红（可能净利润造假，未实际入账） |
   | 资产负债表内部          | 资产总计 = 负债总计 + 股东权益总计（报表最底部两行）         | 完全相等（会计恒等式，必满足）            | 不相等（报表编制错误，或数据造假）                           |

   ### 三表核心科目速查清单（标注 "看什么 + 风险点"，直接对照）

   | 报表名称       | 核心科目            | 核心含义                                    | 看什么（关键判断点）                                         | 风险提示（异常信号）                                       |
   | -------------- | ------------------- | ------------------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------------- |
   | **资产负债表** | 货币资金            | 账上可直接用的现金 + 银行存款               | 1. 金额是否覆盖短期借款（短期偿债能力）；2. 是否有 "受限资金"（附注披露） | 货币资金多但有大量短期借款（可能资金被冻结，需靠借钱维持） |
   |                | 应收账款            | 卖货没收到的钱（给客户的赊账）              | 1. 占营收比例是否超过 30%；2. 增速是否超过营收增速           | 占比过高 / 增速超营收（可能收不回，形成坏账）              |
   |                | 存货                | 囤的原材料、半成品、成品                    | 1. 增速是否超过营收增速；2. 存货周转率（越高越好）           | 增速超营收 / 周转率低（可能卖不出去，需计提跌价）          |
   |                | 短期借款 + 长期借款 | 1 年内 / 1 年以上要还的钱（有息负债）       | 1. 有息负债总额是否超过货币资金；2. 短期借款占比是否过高     | 有息负债超货币资金 / 短期借款占比高（偿债压力大）          |
   |                | 未分配利润          | 累计赚的未分给股东的钱                      | 1. 是否连续增长；2. 与净利润增速是否匹配                     | 连续减少 / 与净利润增速脱节（长期亏损或分红过度）          |
   | **利润表**     | 营业收入            | 卖产品 / 服务的总收入                       | 1. 增速是否稳定（如 10%-20%）；2. 营收结构（核心业务占比是否超 80%） | 增速忽高忽低 / 核心业务占比低（收入不稳定）                |
   |                | 毛利率              | （营收 - 营业成本）/ 营收（产品竞争力指标） | 1. 是否高于行业平均水平；2. 是否连续稳定                     | 毛利率骤降 / 低于行业平均（成本失控或产品降价）            |
   |                | 销售费用            | 广告、渠道、销售团队工资等                  | 销售费用 / 营收比例是否超过行业平均                          | 比例过高（靠砸钱卖货，盈利能力弱）                         |
   |                | 净利润（扣非）      | 扣除偶然收入（如补贴、卖资产）后的核心利润  | 1. 扣非净利润是否为正；2. 增速是否与营收匹配                 | 扣非净利润为负（核心业务不赚钱）                           |
   | **现金流量表** | 经营活动现金流净额  | 核心业务赚的现金                            | 1. 是否为正；2. 是否≥净利润                                  | 为负 / 远低于净利润（利润是纸面富贵，没收到钱）            |
   |                | 投资活动现金流净额  | 买 / 卖资产的现金（如买设备、卖厂房）       | 成长型公司：投资流出为主（扩产能）；成熟公司：流出 / 流入平衡 | 频繁大额流入（卖核心资产补现金流，经营恶化）               |
   |                | 筹资活动现金流净额  | 借钱 / 还钱、融资 / 分红的现金              | 1. 是否靠 "股权融资"（发股票）而非 "债权融资"（借钱）；2. 是否稳定分红 | 频繁借新还旧 / 长期不分红（偿债压力大或利润造假）          |

1. UVIX 期权投资失败反思

   我严重低估了消息面对波动率的影响和高估了自己的心态。
   在我对选择两倍做多恐慌指数波动率ETF时，选择了short call，主要考虑了两点。第一，裸卖看涨期权可以利用时间损耗，而我卖出的期权还有三个月到期，这样就能在吃到足够的时间损耗的同时，也有充足的反应时间。第二，作为两倍做多的波动率指数，其损耗相对较大，从长期来看是向下的，即票面价格总是下跌，因此做空是合适的策略。

   然而，我没有预料到市场已经消化了政府关门和特朗普关于关税的发言等消息，尽管恐慌指数已经上升过了，但突发的美国地方银行信贷欺诈问题又导致VIX从20一路飙升至25，市场瞬间进入极端恐慌状态。

   不过仔细想想，如果再来一次，我可能也很难坚持最初的投资思路。在裸卖看涨期权的情况下，面对无限的亏损，我的心理素质仍然不足。下次我可能会考虑采用straddle或strangle策略，以降低风险。

2. 分散投资最大的作用是降低投资组合的标准差和方差。在选取投资行业或者ETF的时候，选择相关性系数越低的投资组合，可以得到更低的标准差。之前没有意识到，工业和金融的相关性太高了，达到了86%。能源和公共事业的相关性就相对低很多，适合投资 。根据Loretan和English的观点，在高波动性时期，预期的多样化收益可能会减弱，因为资产之间的相关性可能增加，从而削弱低相关性通常提供的保护效果。换句话来说，市场极度恐慌或者极度兴奋的时候，所有的投资标地都会被卖出和买进。

13. 最近突然生出一个强烈的感触：开源精神和买方市场的底层逻辑，似乎从根上就存在一种天然的互斥感。
    学计算机这么多年，"开源" 两个字几乎贯穿了整个学习和成长过程。我们早就习惯了在开源社区里共享资源、交换思路 — 代码公开可查，工具免费可用，开发者们抱团讨论、互相补位，才一步步迭代出如今强大的框架和系统。这种 "开放即共赢" 的理念，早已刻进了技术人的思维里：越多人参与，越能发现问题、优化方案，最终推动整个行业的进步，每个人也能在这个过程中借力成长。
    但踏入买方的领域后，这种认知却被彻底颠覆了。作为市场的流动性提供者，买方的核心逻辑是 "对抗" 而非 "共享"。你的每一套交易策略，本质上都是在和其他市场参与者争夺利润，大家站在同一片赛道上，拼的就是信息差和独家逻辑。一个策略的容量往往是有限的，一旦公开或被太多人知晓，资金涌入就会迅速摊薄收益，甚至让策略直接失效。更极端的是，若核心逻辑被对手吃透，还可能遭到反向狙击，得不偿失。所以在这个行业里，"藏" 成了一种本能：核心策略讳莫如深，关键数据严格保密，就连思路交流也大多点到即止，没人愿意把真正的 "看家本领" 拿出来共享。
    回头想想，这或许就是不同领域的底层规则差异：开源的核心是 "聚合力量"，靠共享实现共赢；而买方的核心是 "抢占先机"，靠稀缺性实现盈利。没有绝对的好坏，只是身处其中，虽然会怀念开源世界的纯粹与开放，也默默把自己的github repo全部设置成private 。 毕竟，不像互联网是做大蛋糕，买方是零和博弈。
